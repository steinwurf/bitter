Bitter Documentation
====================

Bitter is a lightweight header only C++ API, for reading and writing single bit
fields.

The Bitter repository: https://github.com/steinwurf/bitter

The ``examples`` folder contains some programs that demonstrate the usage
of the available API.

If you have any questions or suggestions about this library, please contact
us at our developer mailing list (hosted at Google Groups):

* http://groups.google.com/group/steinwurf-dev
